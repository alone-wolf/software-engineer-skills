# 问题摘要

# 问题描述
当前仓库中部分 skills 的 `skill_version/cluster_version` 为 `1.1.0`，其余仍为 `1.0.0`。缺少明确版本策略会导致“当前集群版本”判定不一致，影响安装后排障、变更审计和文档同步判断。

Medium

- 版本发布说明与问题排查流程

1. 明确版本策略：`skill_version` 可独立递增，`cluster_version` 需全量一致。
